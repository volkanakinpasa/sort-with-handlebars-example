Sorting a list with another list by priority

```
// List
[ {id: 1}, {id: 2}, {id: 3}, {id: 4}, {id: 5} ]
```

```
// List to top
[{5,3}]
```


Output
-------------
| Id     | Normal Sort | Filtered & Sorted   |
| :------- | ----: | :---: |
| 1 | 1 | 5 |
| 2 | 2 | 3 |
| 3 | 3 | 1 |
| 4 | 4 | 2 |
| 5 | 5 | 4 |
